
# CyberRe Grub Theme

## Installation

Download and install the theme (make sure you have `wget`, `unzip` and `sudo` installed):

```bash
# Download the repository
wget https://github.com/Qaddoumi/grub-theme/archive/refs/heads/main.zip -O /tmp/grub-theme.zip

# Extract to /tmp
unzip /tmp/grub-theme.zip -d /tmp

# Navigate to the extracted directory
cd /tmp/grub-theme-main

# Run the installation script
./install.sh

# Clean up
rm -rf /tmp/grub-theme.zip /tmp/grub-theme-main
```

Alternatively, if you already have the repository cloned:

```bash
./install.sh
```
