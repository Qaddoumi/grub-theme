## Installation

Download and install the theme (make sure you have `git` and `sudo` installed):

```bash
# Clone the repository
git clone --depth 1 https://github.com/Qaddoumi/grub-theme.git

# Navigate to the directory
cd grub-theme

# Run the installation script
./install.sh

# Clean up
cd ..
rm -rf grub-theme
```

Alternatively, if you already have the repository cloned:

```bash
./install.sh
```
